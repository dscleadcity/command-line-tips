# Command line tips

Here are some **command-line tips**:

1. Changing folders

``` bash
cd <name-of-folder>
```

2. Displaying folder contents

``` bash
ls
```

3. Creating folders

``` bash
mkdir <name-of-folder>
```

## npm & Yarn

* Adding files

**npm** 
``` bash
npm install <package-name>
```

**Yarn**
``` bash
yarn add <package-name>
```
